# 📊 Data Careers Dashboard

## 📖 About
Este projeto apresenta um **dashboard interativo** construído com **Streamlit**, que explora dados sobre **cargos e salários na área de dados**.  
O objetivo é demonstrar como transformar um dataset em visualizações dinâmicas e acessíveis via web, facilitando a análise de tendências de mercado.

## 🎯 Objetivo
- Criar visualizações interativas sobre salários e cargos em Data Science, Engenharia de Dados, Análise de Dados e áreas relacionadas.
- Mostrar a aplicação prática de **Streamlit** para construção de dashboards.
- Servir como projeto de portfólio em **visualização de dados**.

## 🛠️ Tecnologias Utilizadas
- **Python 3.x**
- **Streamlit**
- **Pandas**
- **Plotly / Matplotlib / Seaborn** (dependendo das visualizações)
- Jupyter Notebook (para tratamento inicial dos dados)

## 🚀 Como Executar
1. Clone este repositório:
   ```bash
   git clone https://github.com/fernando-rgomes/data-careers-dashboard.git
   cd data-careers-dashboard
   
2. Crie e ative um ambiente virtual:
   
python -m venv .venv
.venv\Scripts\activate   # Windows PowerShell
source .venv/bin/activate # Linux/Mac

3. Instale as dependências:
pip install -r requirements.txt

4. Execute o Streamlit:
streamlit run app.py
